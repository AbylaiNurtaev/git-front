# Backend: имя игрока в ленте выигрышей (/club/qr)

На странице клуба [QR](https://git-front-sandy.vercel.app/club/qr) внизу показывается лента последних выигрышей в формате **«Имя выиграл Приз»**. Чтобы выводилось имя, а не номер телефона, бэкенд должен отдавать имя игрока.

## 1. GET /players/recent-wins

Ответ — массив объектов. В каждом объекте желательно передавать имя:

- **`name`** или **`playerName`** — отображаемое имя игрока (например, «Абылай»). Если есть, в ленте будет «Абылай выиграл Fanta».
- **`maskedPhone`** — замаскированный номер (например, «+7 701***1»). Используется, если нет имени.
- **`prizeName`** — название приза.
- **`text`** (необязательно) — готовая строка, например «Абылай выиграл Fanta»; если задана, используется как есть.

Приоритет на фронте: **name** → **playerName** → **maskedPhone** → «Гость».

## 2. Socket.IO: событие `spin`

При выигрыше в комнату клуба отправляется payload. Чтобы в ленте сразу отображалось имя:

- В payload передавайте **`playerName`** или **`name`** — имя выигравшего игрока.
- В **`recentWins`** (массив последних выигрышей) в каждом элементе передавайте **`playerName`** или **`name`** (и при необходимости **`maskedPhone`**, **`prizeName`**).

Пример payload при спине:

```json
{
  "playerName": "Абылай",
  "playerPhone": "+77001234567",
  "recentWins": [
    { "playerName": "Абылай", "prizeName": "Fanta" },
    { "playerName": "Мария", "prizeName": "20 баллов" }
  ],
  "spin": { "prize": { ... } }
}
```

Фронт подставляет имя в строку вида «{имя} выиграл {приз}».
